\# On-Device Activity Recognition



\*\*Purpose\*\*  

Build a lightweight activity recognition system using accelerometer data and classical ML, optimized for efficiency and interpretability.



\## Problem Statement  

Given raw accelerometer and gyroscope signals from wearable sensors, classify user activities such as walking, sitting, standing, etc.



\## Project Structure  

\- \*\*data/\*\*: raw and processed sensor data  

\- \*\*notebooks/\*\*: exploration and modeling notebooks  

\- \*\*src/\*\*: scripts for preprocessing, feature extraction, training, and evaluation  

\- \*\*results/\*\*: saved metrics and plots
